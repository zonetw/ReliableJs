Running the Sample Code
===================================================================================

In each subdirectory for this chapter, you will find index.html or several
index_nn.html files. Open one in your browser to run the tests it contains.



In the ContractRegistry directory
-----------------------------------------------------------------------------------
index.html runs ContractRegistry_tests.js to test ContractRegistry.js


In the ForEach directory
-----------------------------------------------------------------------------------
index_01.html runs forEach_01_tests.js on forEach_01.js.
index_02.html runs the same tests, but on forEach_02.js.


In the OrderedObject directory
-----------------------------------------------------------------------------------
index_01.html runs OrderedObject_01_tests.js on OrderedObject_01.js.
index_02.html runs OrderedObject_02_tests.js on OrderedObject_02.js.
index_03.html and index_04.html run their respective tests on OrderedObject_02j.js.


In the Sequence directory
-----------------------------------------------------------------------------------
index.html runs aspectSequenceProblem_tests.js on aspectSequenceProblem.js.
The file borrowingWithApply.js is illustrative but does not run.


In the Utilities directory
-----------------------------------------------------------------------------------
index.html runs utilities_tests.js on utilities.js.
